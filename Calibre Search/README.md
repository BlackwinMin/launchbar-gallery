# Calibre Search

一套基于 LaunchBar 的 Calibre 书库查询系统，可在任何界面中搜索书籍，免去频频开启 Calibre 软件之苦：

1. 如果书籍在 Calibre 库中，则直接使用 Calibre 内置阅读器中打开；
2. 若是在 DEVONthink 中，则使用 DEVONthink 阅读书籍；
3. 如果在外部文件夹，则用默认阅读器打开（可自行修改）；
4. 在线书城中的图书，直接打开对应页面。
